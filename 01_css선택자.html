        선택자란?
        - 특정 html 태그를 선택할 때 사용하는 기능
        - 태그를 선택하여 원하는 스타일과 기능을 적용 가능하다

모든 선택자

        - html 문서안의 모든 태그를 선택할 때 선택자를 이용 가능

아이디 선택자

html 문서 내에 해당하는 아이디 속성 값을 가진 태그만 선택할 때 사용된다

#아이디명 {color:red;} 와 같은 형태로 사용된다
